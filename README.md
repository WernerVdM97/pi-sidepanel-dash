# pi-sidepanel-dash

> 🤖 This code is an AI-generated proof-of-concept. Use at your own risk.

Session dashboard tab for [pi-sidepanel](https://github.com/WernerVdM97/pi-sidepanel). Merges the standalone `/context` and `/dash` commands into a single sidepanel tab showing goal, session metadata, context budget breakdown, and top tool definitions by token cost.

## Sections

| Section | Shows |
|---------|-------|
| 🎯 Goal | Latest user prompt/objective |
| 🧠 Session | Model, turn count, thinking level |
| 📊 Context | Used / window tokens with percentage bar |
| Breakdown | System prompt, tool definitions, conversation — each with % |
| Top Tools | 5 largest tool definitions by token cost, with active/inactive markers |
